# CS4120 Final Project — AI Text Detection

Binary classification of human-written vs. AI-generated text. Compares fine-tuned transformer models against traditional baselines (TF-IDF, n-grams, embeddings) and neural sequence models (RNN, LSTM, CNN) with custom LR and SVM classifiers, using the HC3 and TuringBench datasets.

## Setup

```bash
pip install -r requirements.txt
```

## Workflow

### 1. Download datasets

**HC3:**

```bash
python scripts/download_hc3.py
```

Saves `data/processed/hc3_train.csv` and `data/processed/hc3_test.csv`.

**TuringBench:**

Download the zip file from [TuringBench]() and unzip into a data folder

```bash
python src/data/turingbench_loader.py
```

Saves `data/processed/turingbench_train.csv` and `data/processed/turingbench_test.csv`.

### 2. Check preprocessing stats

```bash
python scripts/run_preprocessing.py
```

Prints label distributions and average text lengths for each dataset.

### 3. Extract features

```bash
python scripts/run_feature_extraction.py
```

Saves feature matrices to `data/processed/features/hc3/`:

| File                    | Type                      | Description                                 |
| ----------------------- | ------------------------- | ------------------------------------------- |
| `tfidf_*.npz`           | sparse                    | TF-IDF unigrams + bigrams (50k vocab)       |
| `word_ngram_*.npz`      | sparse                    | Word 1–3-grams (50k vocab)                  |
| `char_ngram_*.npz`      | sparse                    | Char 2–4-grams (50k vocab)                  |
| `bert_embeddings_*.npy` | dense `(n, 768)`          | BERT mean-pooled embeddings                 |
| `word2vec_*.npy`        | dense `(n, seq_len, 100)` | GloVe sequence embeddings (used by RNN/CNN) |

### 4. Train and evaluate a single baseline

```bash
python scripts/train_test_save.py --model lr --dataset hc3 --features tfidf
python scripts/train_test_save.py --model svm --dataset hc3 --features word_ngram
```

Saves the trained model to `saved_models/` and metrics to `metrics/`.

### 5. Fine-tune a pretrained model

```bash
python scripts/run_finetune.py
python scripts/run_finetune.py --model distilroberta-base --epochs 3 --max_length 128
python scripts/run_finetune.py --dataset turingbench
python scripts/run_finetune.py --dataset combined
```

The `--dataset` flag accepts `hc3` (default), `turingbench`, or `combined`. Saves the best checkpoint (by validation F1) to `models/`.

### 6. Evaluate a saved pretrained checkpoint

```bash
python scripts/eval_pretrained.py --checkpoint models/distilroberta-base_combined
python scripts/eval_pretrained.py --checkpoint models/distilroberta-base_combined --dataset turingbench
python scripts/eval_pretrained.py --checkpoint models/distilroberta-base_combined --dataset hc3 --max_length 64
```

Prints per-class and per-domain metrics and a confusion matrix for the specified test split.

## Datasets

**HC3** (Human ChatGPT Comparison Corpus) — Q&A pairs from Reddit ELI5, open QA, finance, medicine, and Wikipedia. Labels: 0 = human, 1 = ChatGPT.

**TuringBench** — Articles generated by 19 different language models plus human-written news. Binarized to 0 = human, 1 = AI-generated.
